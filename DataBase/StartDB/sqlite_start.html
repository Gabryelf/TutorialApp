Домашняя библиотека на SQLite

Полное руководство + 15 заданий для смелых. Всё в браузере, без установок.

Начинаем путешествие

Никакой установки SQLite Online Таблица = Excel

Открываем https://sqliteonline.com/ — левое окно для команд, кнопка Run выполнит запрос.

В левом верхнем углу должна быть выбрана SQLite (по умолчанию).

1. Создаём таблицу books

PRIMARY KEY NOT NULL AUTOINCREMENT
CREATE TABLE books (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    title TEXT NOT NULL,
    author TEXT NOT NULL,
    year INTEGER,
    rating INTEGER
);

Скопируй и выполни. После этого таблица пуста, но готова к наполнению.

2. Добавляем книги (INSERT)

-- Первая книга
INSERT INTO books (title, author, year, rating) VALUES ('Маленький принц', 'Антуан де Сент-Экзюпери', 1943, 10);

-- Вторая
INSERT INTO books (title, author, year, rating) VALUES ('1984', 'Джордж Оруэлл', 1949, 9);

-- Третья (без рейтинга)
INSERT INTO books (title, author, year) VALUES ('Война и мир', 'Лев Толстой', 1867);

✨ Чтобы увидеть результат: SELECT * FROM books;

3. Изменяем записи (UPDATE)

-- Исправляем рейтинг 1984 на 10
UPDATE books SET rating = 10 WHERE id = 2;

-- Добавляем рейтинг Войне и миру
UPDATE books SET rating = 8 WHERE id = 3;
Всегда используй WHERE с уникальным id, иначе обновятся все строки!

4. Удаление (DELETE)

DELETE FROM books WHERE id = 1; -- прощай, Маленький принц

После удаления проверь: SELECT * FROM books; — первая книга исчезла.

15 практических заданий

⭐ Каждое задание можно выполнять независимо. Команды приведены готовые, но ты можешь их изменять.

1 Добавь свою любимую книгу
INSERT INTO books (title, author, year, rating) VALUES ('Гарри Поттер и философский камень', 'Дж. Роулинг', 1997, 10);
2 Книга без года
INSERT INTO books (title, author, rating) VALUES ('Слово о полку Игореве', 'Неизвестен', 9);
3 Массовое добавление (две сразу)
INSERT INTO books (title, author, year, rating) VALUES
('Преступление и наказание', 'Ф. Достоевский', 1866, 10),
('Идиот', 'Ф. Достоевский', 1869, 9);
4 Проверь ограничение NOT NULL
INSERT INTO books (title, year) VALUES ('Книга без автора', 2020);
-- ожидай ошибку: поле author не может быть пустым
5 Исправь год Войны и мира
UPDATE books SET year = 1869 WHERE title = 'Война и мир';
6 Подними рейтинг всем книгам с rating < 5
UPDATE books SET rating = 5 WHERE rating < 5;
7 Добавь " (роман)" к названиям Толстого
UPDATE books SET title = title || ' (роман)' WHERE author LIKE '%Толстой%';
8 Проставь рейтинг 7 книгам без рейтинга (NULL)
UPDATE books SET rating = 7 WHERE rating IS NULL;
9 Удали книгу по id (например, id = 5)
DELETE FROM books WHERE id = 5;
10 Удали все книги до 1900 года
DELETE FROM books WHERE year < 1900;
11 Удали книги с рейтингом меньше 4
DELETE FROM books WHERE rating < 4;
12 Удали все книги определённого автора (например, Достоевского)
DELETE FROM books WHERE author LIKE '%Достоевский%';
13 Выбери книги 90-х (1990–1999)
SELECT * FROM books WHERE year BETWEEN 1990 AND 1999;
14 Топ книг по рейтингу (сортировка)
SELECT title, author, rating FROM books ORDER BY rating DESC;
15 Удали книги с названием короче 3 символов
DELETE FROM books WHERE LENGTH(title) < 3;

Бонус: добавляем столбец price

ALTER TABLE books ADD COLUMN price REAL;

-- Теперь можно проставить цены, например:
UPDATE books SET price = 499.99 WHERE id = 2;
REAL для дробных чисел
Правило выживания: Перед выполнением UPDATE или DELETE без WHERE — подумай дважды. Лучше сначала сделай SELECT с тем же условием, чтобы увидеть, что именно ты собрался менять.
Твой первый шаг в базы данных. Пробуй, ошибайся, исправляй — это нормально.
