## Задание

Спроектируйте 4 таблицы, которые дополняют структуру GitHub.

Для каждой таблицы:
1. Расчитайте **колонки** отталкиваясь от логики
2. Определите **колонки** (аргументы)
3. Заполните **параметры** типы данных в пустом месте под колонкой (Integer, Varchar, Time, Text)
4. Выделите **первичный ключ** (PK)
5. Укажите **внешние ключи** (FK) и связи с таблицами выше

|    id    |       name      |     description     |      owner_id      |    created        |
|--------|-------------|----------|------------|------------|
|   AI  PK   |       Not Null      |     Null     |       FK     |    Not Null        |
|  INTEGER      |     Varchar        |    Text      |     INTEGER       |     Time       |
