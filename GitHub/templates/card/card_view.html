Загрузка карты

Как использовать:

  • Создайте карту в BattleScript Creator
  • Скачайте файл .md
  • Загрузите его здесь для просмотра
  • Проверьте, как карта выглядит в игре
  • Для аватаров используйте: https://i.pravatar.cc или https://picsum.photos
Перетащите файл .md сюда
или нажмите для выбора файла
